  ## THE GOSSIP PROJECT 

  <h2>Contenu</h2>

  ### 1- 8 models

  ### 2- 8 migrates correspondants

  ### 3- Un beau seed pour mélanger tout ça

## Etapes d'utilisation :



Pour lancer le programme une fois gitcloné :

```

   $> bundle install

```
Ensuite : 

```

   $> rails db:migrate

```
Enfin : 

```

   $> rails db:seed

```
